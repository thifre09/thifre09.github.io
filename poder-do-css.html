CSS (Cascading Style Sheets ou Folhas de Estilo em Cascata) é uma linguagem de estilo usada para descrever a apresentação de um documento escrito em HTML ou em XML. O CSS descreve como elementos são mostrados na tela. Por exemplo: ele determina a cor de um texto, o tamanho dele, uma sombra, o espaçamento entre elementos, e muito mais. Nessa página, veja como o css é importante para estilizar uma página, ao apertar no botão no fundo da página, todo o css da página(com excessão do css do botão) será desativado.
